# Better Rewind

Hold the vinyl button to rewind through a track at 1x speed, like you're doing a boiler room set. Release to stop and resume playback from wherever you landed.

Based on [Reeeeewwwwinnnddd](https://github.com/NickColley/spicetify-rewind) by [Nick Colley](https://github.com/NickColley).

## How it works

- **Hold** the vinyl button to rewind continuously at real-time speed
- **Release** to stop rewinding and resume playback from that position
- Rewind sound effect plays and loops while held
- Automatically stops when it reaches the start of the track
- Works with mouse and touch input
